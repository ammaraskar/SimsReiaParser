# ReiaTool

A simple tool for end-users to encode video/gifs as `.reia` files or to decode
existing `.reia` files.

## Development

### Running from Source

`poetry run python reiatool/reiatool.py`

### Formatting 

`poetry run black .`

### Packaging for Distribution

`poetry run pyinstaller --windowed reiatool/reiatoo.pyw`
